Mustang-F100 msp430 firmware update tool Linux /Windows version
2.Linux 
	a. extract the iei_f100msp430_updater_v1.0.0.20200706.tar.gz file
	b in ubuntu , click mouse right button to extract here
	c. enter file folder 
	d. command : $ sudo ./iei_fw_flash.sh <Firmware txt file>
	e. example $ sudo ./iei_fw_flash.sh Mustang-F100_MSP430_U65_V1030_A_20181127.txt
	
3.Windows
	a. in windows 10 extract iei_f100msp430update_win_v1.0.0.20200706.7z
	b. execuate iEiMsp430Updater.exe
	c. Device Scan
	d. select firmware txt file to update the MCU
